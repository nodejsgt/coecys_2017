Objetivos del Taller

El objetivo final de este Taller, es brindar más que una introducción, es demostrar el verdadero poder que puede tener un Framework FrontEnd como Angular, para la creación de aplicaciones Web en la Industría actualmente.

Así mismo, busca poder proporcionar a los asistentes del COECYS, las herramientas y conocimientos necesarios para poder crecer profesionalmente en el ámbito de desarrollo Web. De igual forma, los asistentes crearán una aplicación que podrán presentar como parte de su portafolio profesional
